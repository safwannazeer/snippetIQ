# 🚀 SnippetIQ

**SnippetIQ** is a smart code snippet management platform with AI-powered assistance. It helps developers store, organize, search, and reuse code efficiently while also generating and explaining code using AI.

---

## 📌 Features

- 🔐 User Authentication (Register/Login)
- 📝 Create, Edit, Delete Code Snippets
- 🏷️ Tag-Based Organization
- 🔍 Search & Filter Snippets
- 🤖 AI Code Generation (Qwen API)
- 💡 AI Code Explanation
- 🖥️ Monaco Editor (VS Code-like experience)

---

## 🧠 Problem Statement

Developers often rewrite the same code repeatedly and struggle to manage reusable snippets efficiently. Existing solutions lack proper organization and AI assistance.

---

## 💡 Solution

SnippetIQ provides a centralized platform where developers can manage snippets and use AI to generate and understand code instantly.
